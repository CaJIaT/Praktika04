# Changelog

Все заметные изменения проекта будут документироваться в этом файле.

## [0.3.0]
### Добавлено
- Реализован регистр накопления «Методическая работа преподавателей» с расчётом баллов
- Добавлена табличная часть в документ «Учёт методической работы» с привязкой к видам работ
- Добавлен простой отчёт «Реестр методической активности преподавателей» (СКД)

### Изменено
- Переименованы некоторые реквизиты для большей ясности
- Добавлены проверки заполнения обязательных полей в документах

## [0.2.0]
### Добавлено
- Документ «Распределение учебной нагрузки» с табличной частью
- Регистр сведений «Преподаваемые дисциплины» (периодический)
- Связи между справочниками: Специальности → Группы → Студенты
- Базовые управляемые формы для всех основных объектов

### Исправлено
- Исправлена ошибка автозаполнения преподавателя в нагрузке

## [0.1.0]
### Добавлено
- Начальная структура конфигурации
- Справочники: Сотрудники, Студенты, Учебные группы, Специальности, Дисциплины, Виды методической работы
- Первый документ «Учёт методической работы» (с реквизитами и табличной частью)
- Простейшая командная панель и разделы интерфейса

**Начало проекта** — демонстрационная конфигурация для учебной практики.
